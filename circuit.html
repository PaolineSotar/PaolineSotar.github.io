SHORT-CIRCUIT

Experimental practice starting from the notion of Short-circuit. The idea was based on daily encoded objets (like the bus ticket, the SD card, the Jacquard weaving loom) and to implant on it a virus, that will break the code. The format, the texture, the weight of each objects has been represented with different kinds of papers. The code become an ornamental pattern, a secret language, now playful when the pattern is cracked by the bug, the troublemaker element. Graphic forms’ codes has been reinterpreted to make new short-circuits, not a virtual but a concrete one.

Work with laser cutting machine on paper



FR / COURT-CIRCUIT

Pratique expérimentale à partir de la simple notion de court-circuit. L’idée était de se baser sur des objets de notre quotidien cryptés comme le ticket de métro, la carte SD, la puce, les métiers à tisser Jacquards, et d’y implanter un bug, qui viendrait court-circuiter le code. Le format, la texture, le poids de chaque objet ont été représentés avec différents papiers. Le code devient ici un motif ornemental, un langage caché qui devient ludique lorsque la série est rompue par le bug, l’élément perturbateur. Les formes graphiques des codes ont été réinterprétées pour court-circuiter les codes déjà existants.
